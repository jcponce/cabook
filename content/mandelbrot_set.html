The Mandelbrot Set

Introduction

The Mandelbrot set is certainly the most popular fractal, and perhars the most popular object of contemporary mathematics of all. Since Benoît Mandelbrot (1924-2010) made his extraordinary experiment in 1979, it has been duplicated by tens of thousands of amateur scientists around the world (including myself).

The Mandelbrot set is generated by iteration, which means to repeat a process over and over again. For the Mandelbrot set, the functions involved are quadratic polynomials and have the form \(f(z) = z^2 + c\), where \(c\) is a constant number.

Although it is defined by a simple rule, it posseses interesting and complex propertires which can be seen graphically.

Constructing the Mandelbrot Set

Here is how the Mandelbrot Set is constructed. Take a starting point $z_0$ in the complex plane. Then we use the recursive formula $$z_{n+1}=z_{n}^2+z_0$$ to obtain a sequence of complex numbers $\{z_n\}$ with $n=0, 1, 2, \ldots$. The points $z_n$ are said to form the orbit of $z_0$, and the set $M$ is defined as follows:

If the orbit $z_n$ fails to go to infinity, we say that $z_0$ is contained within the set $M$. If the orbit $z_k$ does go to infinity, we say that the point $z_0$ is outside $M$.

Take, for example, $z_0=1$. Then we have \[ \begin{array}{rcl} %\hline %\text{ } & z_{n+1}=z_{n}^2+z_0 \\ %\hline z_0 &=& 1 \\ %\hline z_1 &=& 1^2 + 1 = 2 \\ %\hline z_2 &=& 2^2 + 1 = 5\\ %\hline z_3 &=& 5^2 + 1 = 26 \\ %\hline z_4 &=& 26^2 + 1 = 677 \\ %\hline &\vdots& \end{array} \] As you can see, in this case $z_n$ just keeps getting bigger and bigger. Thus 1 is not in the Mandelbrot set. But if we choose different values for $z_0$ this won't always be the case. Now consider the value $z_0=i$. In this case, we have the following values: \[ \begin{array}{rcl} %\hline % \text{ } &=& z_{n+1}=z_{n}^2+z_0 \\ %\hline z_0 &=& i \\ %\hline z_1 &=& i^2 + i = -1 + i \\ %\hline z_2 &=& (-1+i)^2 + i = (1*2i-1)+i = -i\\ %\hline z_3 &=& (-i)^2 + i = -1+i \\ %\hline z_4 &=& (-1+i)^2 + i = -i \\ %\hline &\vdots& \end{array} \] It is clear that in this case further iterations will just repeat the values $−1+i$ and $−i$. All of these complex numbers lie within distance 3 of the origin. So they stay in a bounded subset of the plane; they do not run out to infinity. So the number $i$ is in the Mandelbrot set.

It is great fun to calculate elements of the Mandelbrot Set and to plot them. The resulting set is endlessly complicated. And for this purpose we can use the power of the computer.
